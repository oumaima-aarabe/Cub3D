## Controls

Controls for movement and rotation are:

- `W`: move forward
- `S`: move backward
- `A`: strafe left
- `D`: strafe right
- `Q`: rotate left
- `E`: rotate right
- `mouse`: rotate by moving the mouse (bonus only)
